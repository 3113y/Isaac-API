# 文件 "translations.xml"

此文件用于存储可能的游戏翻译包映射。主要用于日语和韩语。

???+ bug
    该文件似乎没有实际作用！

**Resource-Folder**{: .xmlInfo .red}：将此文件放入模组的resources文件夹无效！

**Content-Folder**{: .xmlInfo .red}：将此文件放入模组的content文件夹无效！
